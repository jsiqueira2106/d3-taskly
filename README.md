# d3-taskly

Modern offline-first Task Management App built with **Kotlin + Jetpack Compose**.

**Features**
- Clean Architecture (MVVM + Repository pattern)
- Fully offline with Room + WorkManager background sync
- Beautiful Material 3 UI with animations
- Dark mode + dynamic colors
- CI/CD pipeline with GitHub Actions

**Tech Stack**
- Kotlin 100% + Jetpack Compose
- Hilt, Coroutines/Flow, Room, WorkManager
- Offline-first with caching and conflict resolution
